# Smart University Ecosystem

This repository contains the documentation and source code for the Smart University Ecosystem project.

## File Descriptions

This project is organized into several files and directories. Here is a brief overview of each:

*   **[REQUIREMENTS.MD](./REQUIREMENTS.MD):** This file outlines the functional and non-functional requirements of the system. It details the features and constraints that guide the development process.

*   **[USE_CASES.md](./USE_CASES.md):** This document describes the various user interactions with the system. It includes use case diagrams and detailed descriptions for each user role (Student, Teacher, Registrar, Admin).

*   **[INTERVIEW_DOCUMENTATION.md](./INTERVIEW_DOCUMENTATION.md):** This file contains the raw notes and findings from the interviews conducted with stakeholders (teachers, students, and administrators). It serves as the primary source for the requirements elicitation process.

*   **[final.md](./final.md):** This document provides a consolidated summary of the project requirements, including interview notes, classified requirements, and key findings.

*   **[clean.md](./clean.md):** This file contains a cleaned-up and structured version of the assignment requirements and guidelines.

*   **[tacher.md](./tacher.md):** This file contains the interview notes and questions specifically for the teacher stakeholder.

*   **[ai_backend/](./ai_backend/):** This directory contains the Python source code for the face recognition backend. It includes the main application, tests, and image assets.
